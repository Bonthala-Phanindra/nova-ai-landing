# NovaAI – AI Writing Assistant Landing Page

**Live Demo:** https://nova-ai-landing.vercel.app

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui components
- Vercel v0 (used to generate 90%+ of the UI)

## Sections built using Vercel v0
- Hero Section  
- Features Section (3 cards)  
- Testimonials Section  
- Contact / CTA Form  
- Footer  

All UI components were generated directly from v0.dev prompts and copy-pasted into the project.

Fully responsive • Mobile-first • Deployed on Vercel

Built for the Vercel v0 + Next.js task submission.
